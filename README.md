i3ircterm
=========

In-space IRC terminal

Prerequisites
=============

* [py-thermal-printer](https://github.com/luopio/py-thermal-printer)
  * [Python Imaging Library](http://www.pythonware.com/products/pil/) 


Pins
====

| P9 Pin | Purpose | Device |
|--------|---------|--------|
| 11     | UART4 RX| Printer|
| 13     | UART4 TX| Printer|
